Full platform rebuild

$25,000 · milestone-based
This is a separate engagement — a complete redesign and rebuild of Tack on a modern, serverless stack. No hard deadline. Milestones are paid as delivered.

A ground-up redesign and rebuild of the entire Tack platform. New information architecture, new UI, new tech stack. Everything learned from running the existing app — what works, what's confusing, what's missing — gets built into the new version from the start.

The new version runs on fully managed infrastructure at $59–104/month with no server maintenance. One codebase, one language, one deploy. No dev shop dependency.

Includes the complete reporting feature as the centerpiece — filters, drill-downs, historical averages, billable hours for both employee and client use cases, PDF/CSV export. Plus data migration from the existing app so users transition seamlessly.
